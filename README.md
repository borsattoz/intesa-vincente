# intesavincente.it
[intesavincente.it](https://intesavincente.it/) Ã¨ un simulatore per il browser in TypeScript de "L'intesa vincente", penultimo gioco del programma Rai [Reazione a catena](https://www.raiplay.it/programmi/reazioneacatena).

## Sviluppo
Installare le dipendenze:
```sh
npm i
```
Eseguire in una finestra di terminale:
```sh
npm run build-watch
```
Contemporaneamente, eseguire in un'altra finestra di terminale:
```sh
npm start
```
